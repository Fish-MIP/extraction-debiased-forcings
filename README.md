# Description

This repository contains Python script to extract climate forcings for the OSP protocols.

Simulation were originally run on Datarmor by Matthieu Lengaigne.

The scripts provided here extract the IPSL list of variables listed in the protocol (https://protocol.isimip.org/#/ISIMIP3b/marine-fishery_regional/marine-fishery_global/31-forcing-data) and convert them in the right units and format
